# 🦔 Chat with the Indian Pangolin

An interactive educational web application developed for the **Species Advocacy Competition** to creatively advocate for the conservation of the **Indian Pangolin (*Manis crassicaudata*)**, one of the world's most trafficked and endangered mammals.

Instead of presenting conservation information as static text, this project gives the pangolin **its own voice**. Users interact by asking questions, and the pangolin responds through **speech synthesis**, creating an engaging experience that encourages empathy and awareness.

---

## 📖 Project Overview

Conservation education is often delivered through posters, reports, or presentations that may not fully engage audiences. This project explores a more interactive approach by allowing users to "chat" with the Indian Pangolin.

The application presents the pangolin as if it were speaking for itself—sharing information about its habitat, diet, ecological importance, threats, and conservation challenges. By combining interactive questions with voice responses, the project aims to build an emotional connection between users and this endangered species.

This project was created as part of the **Species Advocacy Competition**, demonstrating how technology can be used to promote wildlife conservation through immersive digital storytelling.

## 💡 How It Works

The application presents users with a series of questions related to the Indian Pangolin.

When a question is selected:

- The pangolin displays an answer.
- The response is spoken aloud using the browser's Speech Synthesis API.
- Additional related questions are suggested to encourage further exploration.

Users can also:

- View habitat information
- Watch an educational video
- Learn quick facts using the side information buttons

---

## 🌱 Educational Impact

The project aims to transform conservation awareness into an interactive experience.

Rather than simply reading facts, users hear the pangolin "speak" about its struggles, threats, and importance within the ecosystem. This storytelling approach helps foster empathy and encourages greater public awareness of wildlife conservation.
